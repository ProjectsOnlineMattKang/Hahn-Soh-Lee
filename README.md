# Hahn-Soh-Lee
Korean Romanization Project

This system developed over ten years of reflection on the limitations of existing Korean romanization
for English speakers. The framework was refined and formalized in collaboration with AI (Claude,
Anthropic, 2026). The linguistic instincts, observations, and core design decisions originate from a
Korean American perspective — a viewpoint that existing official systems do not adequately represent.

# Overview

Hahn-soh-lee (한-소-리, meaning “Korean sound”) is a romanization system designed from the ground
up for English-speaking readers and speakers. Existing systems such as the Revised Romanization of
2000, while an improvement over earlier attempts, were developed primarily by native Korean
speakers and reflect assumptions that do not translate cleanly to English phonetics. The result is a
system that frequently misdirects English mouths toward incorrect pronunciations.
Hahn-soh-lee prioritizes four goals above all others:
* An English speaker should arrive at an approximately correct pronunciation on the first
attempt, without prior instruction.
* The system must be internally consistent — one symbol, one sound, no exceptions within the
core rules.
* Reversibility — any HSL word can be reconstructed back into Hangul exactly, because the
mapping is one-to-one in both directions.
* Keyboard Compatibility — HSL sequences map directly to Korean keyboard input order. An
English speaker who knows HSL can type Korean by following the romanization sequence. The
Hangul block assembles automatically.

 b + ah + n → ㅂ + ㅅ + ㄴ → 반

These principles work together. Revised Romanization and McCune-Reischauer are one-way streets —
you cannot reliably reconstruct the original Hangul from their romanized forms, and they do not map
to keyboard input order. Hahn-soh-lee is lossless in both directions and directly actionable at the
keyboard.

훈 → h + oo + n → hoon (ㅂ = h, ㅜ = oo, ㄴ = n)

hoon → h + oo + n → 훈 (fully reversible)
